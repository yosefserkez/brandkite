# Attribution

The skills in this corpus are original distillations informed by the
following openly licensed works. No file is a verbatim copy; ideas and
specific detector rules were adapted with attribution as required.

- **Anthropic Skills — frontend-design, canvas-design**
  https://github.com/anthropics/skills — Apache-2.0.
  Distributional-convergence framing, the three AI-default looks,
  typography-carries-personality, two-pass plan-then-critique process.
- **Impeccable** by Paul Bakaus
  https://github.com/pbakaus/impeccable — Apache-2.0.
  Anti-pattern detector rules (overused fonts, cadence tells, em-dash
  overuse, aphorism pattern, buzzword list additions, slop-look tells).
- **Vercel Web Interface Guidelines**
  https://github.com/vercel-labs/web-interface-guidelines — MIT.
  Accessibility/interaction/typography floor used in the dev-side skill.
- **Microsoft Skills — frontend-design-review**
  https://github.com/microsoft/skills — MIT.
  Three quality pillars and review-severity framing.
- **LottieFiles motion-design-skill**
  https://github.com/lottiefiles/motion-design-skill — MIT.
  (Reserved for Phase B motion choreography.)
